# Contributing to HRECOS Dashboard

Thank you for your interest in contributing to the HRECOS Dashboard!

## How to Contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/my-feature`
3. Make your changes
4. Test your changes locally (see README.md for setup instructions)
5. Commit your changes: `git commit -am 'Add new feature'`
6. Push to the branch: `git push origin feature/my-feature`
7. Submit a pull request

## Pull Request Checklist

- [ ] Code follows the project's style guidelines
- [ ] Tests pass locally
- [ ] Docker build is successful
- [ ] Documentation is updated
- [ ] Commit messages are descriptive

## Code Style

- Python: Follow PEP 8 guidelines
- JavaScript/React: Use consistent formatting
- HTML/CSS: Maintain accessibility standards

## Reporting Issues

Please use [GitHub Issues](https://github.com/Strobingn/hrecos-dashboard/issues) to report bugs or request features.

## Questions?

Feel free to open a [GitHub Discussion](https://github.com/Strobingn/hrecos-dashboard/discussions) for questions.
